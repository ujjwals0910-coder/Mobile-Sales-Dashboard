# Mobile-Sales-Dashboard
This project features an interactive Power BI dashboard designed to analyze mobile phone sales data across different brands, models, regions, and time periods. The dashboard helps uncover sales trends, customer preferences, and revenue performance to support business insights.

Key Features
- Brand-wise and model-wise sales analysis
- Revenue and quantity sold tracking
- Region-wise and city-wise performance
- Monthly and yearly sales trend analysis
- Interactive filters and slicers for dynamic exploration
- KPI indicators for quick performance overview

Tools & Technologies
- Microsoft Power BI
- DAX Functions
- Data Cleaning & Data Modeling
- Data Visualization

Use Case
This dashboard can be used for:
- Sales performance monitoring
- Market and demand analysis
- Identifying top-selling brands and models
- Business decision support
